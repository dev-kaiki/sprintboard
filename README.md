# SprintBoard

Trello-like Kanban focused on UX

## Links
- Live (Vercel): https://dev-kaiki-sprintboard.vercel.app
- Swagger (Render): https://dev-kaiki-sprintboard-api.onrender.com/docs
- Health: https://dev-kaiki-sprintboard-api.onrender.com/health

## One-click deploy

API (Render Blueprint):
https://render.com/deploy?repo=https://github.com/dev-kaiki/sprintboard

WEB (Vercel):
https://vercel.com/new/clone?repository-url=https://github.com/dev-kaiki/sprintboard&project-name=dev-kaiki-sprintboard&repository-name=sprintboard&root-directory=apps/web

Vercel ENV:
NEXT_PUBLIC_API_URL=https://dev-kaiki-sprintboard-api.onrender.com

## Local (Windows)
Run:
powershell -ExecutionPolicy Bypass -File .\dev.ps1
